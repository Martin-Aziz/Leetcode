## âœ… Optimal Solution

<details>
<summary>ğŸ” DFS Solution</summary>

```python
def cloneGraph(node: 'Node') -> 'Node':
    if not node:
        return None
    
    visited = {}
    
    def dfs(node):
        if node in visited:
            return visited[node]
        
        clone = Node(node.val)
        visited[node] = clone
        
        for neighbor in node.neighbors:
            clone.neighbors.append(dfs(neighbor))
        
        return clone
    
    return dfs(node)
```

**Time:** O(N + E) - visit each node and edge once  
**Space:** O(N) - hash map + recursion stack

        return None
